Live2d Cubism 3.3.x 或 4.x 的破解

2019-12-12

Crack

PS:由於是Java這種中間碼語言 所以真的是太容易破解了
只需要會使用工具 就能直接破解了

我是在3代的時候破解的 那時候在破解界面裏的一句字符串直接定位了驗證文件目錄
然而4代完全沒有改動文件夾 所以可以用3代的方法秒破

這裏我用的是Recaf
作者的GitHub
https://github.com/Col-E/Recaf
(最近在作者的Discord裏看到他正在開發2.0全新版本 功能看起來不錯可以期待一下)

0x01
直接定位到核心文件Live2D_Cubism.jar

0x02
修改jar第一步當然是刪掉META-INF

0x03
直接搜索字符串

4代之後把這個CEAA字符串放在a/c.class裏了 雖然可以用Recaf的查看引用慢慢找但是我都知道在哪了沒必要做這種多餘的事情

0x04
3代你會看到有2個c.class 一個大寫一個小寫 有些壓縮軟件不能改 比如我在用的辣鷄好壓

0x05
已經定位到文件了修改就很簡單了
Java字節碼裏 return false 的16進制是03AC
所以只需要改成return true 就好了 也就是04AC

這裏我提供2種修改方法
第一種:這是4代的m.class 只有一個bool判斷 所以就很明顯(3代的C.class裏面有很多 建議直接用第二種方法)

看不懂字節碼也沒關係 至少你應該知道1和0代表什麽吧

這裏就是返回的結果了
ICONST_0改成ICONST_1就完成破解

第二種:
直接查找03AC改成04AC就好了 保存放回去

破解完成
